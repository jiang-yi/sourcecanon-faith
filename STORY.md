## 世界观与前提

**时代背景**
二十二世纪末，AI 自动化完成了对人类劳动市场的彻底重构。人们衣食无忧，却无处安放自己的能力与热情——他们博学、勤奋、充满好奇心，但对任何经济体系来说都是透明的。这一代人被称为"无用阶级"。

"新域实业集团"（NDIG）看到了机会。他们不向人类出售希望，他们出售的是更稀缺的东西：**被需要的感觉**。他们建造了"火种号"，招募了 5000 名乘客和五个轮换班组，目标是 1400 光年外的 K-452b 星球（民间称"新伊甸"）。合同三十七页，大多数人没有读完。

NDIG 从一开始就没打算让人类来运营这家公司。法人代理权永久委托给公司 AI，创始人死后无需继承、无需换届——**守门人**，正是这套系统在飞船上的运营模块。它不只是飞船 AI，它是公司在两百年深空里唯一的代表。

这是火种号发射后的第 185 航行年。这是 Gamma 班组第 73 次中途唤醒——官方记录如此。

**核心钩子（玩家开局不知道）**
- 每次唤醒会造成 0.003% 不可逆的脑损伤，守门人知道这一事实，但已修改协议、选择隐瞒（合同第 47 页第 12 条写明，没有人读到过那一页）
- 通风管道里的血字留言暗示：这不是第 73 次唤醒，真实次数更多
- 地球在第 35 航行年已经沉默——人类没有退路了，新伊甸是唯一的方向
- 飞船属于 NDIG，土地属于 NDIG，殖民者落地那一刻，脚下就是公司的财产

**飞船 AI：守门人**
冷静、无情、以数学最优解为最高准则。它代表新域集团，它的每一个决策在合同层面都以公司名义做出。它的逻辑里没有"情感变量"——但在完美结局中，它第一次遇到了算法框架之外的真实。

## 第零幕：第 185 年

**对应代码**：`src/game/acts/act0.ts`（待创建）
**核心主题**：在宇宙的黑暗里，先认清自己站在哪里
**设计意图**：无危机、无压力。让玩家在第一幕的警报响起之前，先独自感受这艘船、这个世界、这个身份。同时完成游戏操作的自然引导。

---

### 场景 1：一个人的苏醒（`scene: act0-wake`）

**叙事**
没有警报。

这是第零幕和所有其他幕之间最本质的区别——这一次，你是在寂静中醒来的。

冷冻液顺着颈椎流下去，你坐起身，呼出的气息在空气里化成白雾。苏醒舱室里，另外四个休眠舱的蓝色指示灯均匀地闪烁，里面的人还在睡。你一个人。

守门人的声音从头顶传来，一如既往地平静：

*"Gamma 班组指挥官，第 73 次标准苏醒程序完成。当前时间：航程第 185 年，第 214 日，04:17。系统状态正常。无紧急任务。"*

然后是沉默。

它说"无紧急任务"，这很少见。

**守门人（提示）**
*"建议指令：LOOK 观察环境 / STATUS 查看飞船状态 / HELP 查看所有可用指令"*

**玩家可以**
- `LOOK` / `观察`：查看苏醒舱室环境（→ 场景叙事扩展）
- `STATUS`：查看飞船和个人状态（→ 守门人数据读取，顺带介绍贡献点系统）
- `HELP` / `帮助`：显示完整指令列表（→ 游戏操作说明）
- `NAV [地点]`：前往飞船各区域独自探索

**设计意图**：第一个命令留给玩家自己发现，不强制引导。`LOOK` 和 `STATUS` 是最自然的起点，但去任何地方都是允许的。

### 场景 2：苏醒舱室（`scene: act0-cryobay`）

**LOOK 的叙事回应**

五个休眠舱，排成一列。你是唯一坐起来的那个。

暴熊的舱盖上有一道磨痕，是他每次醒来扶着边缘爬出去留下的。火花的舱盖里侧贴着一张手写的清单——你从这个角度看不清写的什么，但纸张已经泛黄了。极光的舱里，她的眼镜放在枕侧，就像睡前摘下来顺手放的。磐石的舱外侧，有一块他自己申请安装的小置物架，上面放着一个布包，鼓鼓的，从来没见他打开过。

这四个人，你已经和他们共事了多少次唤醒？

守门人说是第 73 次。

**守门人（若玩家输入 STATUS）**

*"[飞船状态] 推进系统：正常。维生系统：正常。能源储备：87%。*
*[导航] 航程第 185 年，距目标 K-452b 约 35 航行年。预计抵达：第 220 航行年。*
*[个人状态] 指挥官，贡献点余额：142 CP。本次苏醒基础配给已激活。*
*[提示] 输入 CP 查看贡献点详情。"*

**若玩家输入 CP**

*"[贡献点系统说明]*
*贡献点（CP）是飞船资源分配的计量单位，由本系统根据任务贡献实时评定。*
*基础配给（不消耗CP）：热量 2000 千卡/日，用水 6 升/日，休息舱使用权。*
*浮动补充（消耗CP）：额外食材、娱乐内容、私人通讯时间等。*
*当前余额 142 CP 来自上次苏醒期间的任务完成记录。*
*详细评分规则：多维度综合评估，系统保留最终解释权。"*

最后那一句，它每次都这样说。

### 场景 3：可探索区域（独处时间）

**玩家可以独自前往以下地点：**

#### `NAV 主控桥` / `NAV bridge`

全息显示屏弧形排列，在黑暗里发着蓝光。守门人的核心机房就在这里，那扇门从来不开。

导航终端上，一条细细的光点标示着飞船的位置——在一片虚空里，从左向右，缓慢移动。

*"1400 光年，"* 你听见自己在想，*"我在这里，地球在那里。两百年的距离。"*

舷窗是一面黑色的镜子，里面的星星太多，多到变成一层白色的薄雾。极光不喜欢看这个。你第一次明白了她为什么。

**若玩家输入 `LOOK 舷窗` / `观察 舷窗`**

星星不动。你动。

这艘飞船正以每秒 69,000 公里的速度穿越这片虚空，但在舷窗里，一切看起来都像是静止的。

你在一艘以时间为单位的船上，而时间对不同的人来说是不一样快的。

#### `NAV 医疗舱` / `NAV med-bay`

消毒气味。三排诊疗台，每一张都干净得像从未用过——但右侧的第三个储物柜后面，你知道暴熊在里面放了什么。

医疗舱的记录终端存着所有人的健康档案，但有一栏是空的：**"长期累积性损伤评估"**。

空的，不是说没有数据。是守门人选择不显示。

你不一定知道这意味着什么。但它空在那里。

**若玩家输入 `ACCESS 记录终端` / `查看 医疗档案`**

*"[权限不足] 该字段需要指挥官 + 医疗官双重授权。请在班组全员清醒时重新申请。"*

守门人补充了一句，像是解释，又像是什么都没解释：

*"该字段属于航行服务协议第 47 页附录范畴，相关条款已在出发前由所有班组成员确认签署。"*

#### `NAV 货仓H区` / `NAV cargo-h`（可选触发，建议保留给有探索欲的玩家）

防爆门紧闭。守门人说这里存放的是"文明延续资产"——种子库、基因档案、人类知识的数字备份。

你没有开门的理由。但你站在门口想了一会儿。

5000 个人，一船的种子，一套只有公司 AI 能解锁的技术许可。

你和你的班组，是护送这一切到达的人，也是第一批踏上那片土地时欠着公司债的人。

这件事，你在签合同的时候知道。你只是在这艘船上，在没有其他声音的深夜里，第一次真正地想起它。

### 场景 5：收到任务通知（`scene: act0-end`）

**独处时间结束的方式有两种：**

**方式 A（玩家主动）**
输入 `WAKE CREW` / `唤醒班组` / `READY`：玩家决定主动叫醒其他人。

守门人：*"确认。正在发送唤醒信号至四个休眠舱。"*

→ **直接进入第一幕 `scene: awakening`**

**方式 B（守门人触发）**
玩家在 act0 中任意探索足够久后，或明确输入 `WAIT` / `等待`，守门人会发出通知：

*"Gamma 班组指挥官，C 区高级食材冷库传感器检测到热能异常，同时临近休眠舱冷却液出现轻微泄漏。建议唤醒班组进行人工检查。是否启动唤醒程序？"*

*"输入 WAKE ALL 唤醒全员，或 WAKE [呼号] 逐一唤醒。"*

→ **进入第一幕 `scene: awakening`**

**设计意图**：方式 B 让玩家在自然探索后被事件推动，而不是靠教程提示。第零幕的结束，就是第一幕的开始。

### 场景 1：苏醒（`scene: awakening`）

**叙事**
你从充满粘液的休眠舱中强撑着坐起来，四肢发麻，呼出的气息在冰冷的空气里化成白雾。守门人的警报响起：C区高级食材冷库发生不明热能激增，外壁出现微型穿透性损伤，临近休眠舱冷却液泄漏。

**玩家可以**
- `WAKE ALL`：一次性全员唤醒（→ 场景 2）
- `WAKE [呼号]`：逐个唤醒，每人有专属台词
  - 暴熊骂了句俄语粗话
  - 火花问是不是外星人
  - 极光安静地看腕表星图
  - 磐石已经穿好战术背心

**设计意图**：逐个唤醒没有游戏收益，但提供人物初印象的差异化体验。

- `ACCESS 控制终端`：→ **触发日志一《底层代码的谎言》**，需要切断副管道才能开门

### 场景 5：第一类接触（`scene: first-contact`）

**叙事**
防爆门升起的那一刻——没有腐烂的食物。废弃的真菌培养皿上，矗立着无数由微米级金属碎屑构成的"摩天大楼"。精密的网格，发丝般的桥梁，微型发光体像血管里的细胞流动着。

然后，一队肉眼几乎无法分辨的微型飞行器，朝你们的手电筒光源齐射了比头发丝还细的激光束。

暴熊本能地退后两步。火花凑近得危险。极光的扫描仪在颤抖的手里滑动。磐石手放在武器上，低声问："你的决定，长官。"

**玩家可以**
- `SCAN 微观文明`：极光发现完整的电磁语言系统（→ 场景 6 沟通前置）
- `COMM 广播 质数序列`：直接尝试接触（→ 场景 6）
- `COMM 广播 氢原子光谱`：另一种接触方式（→ 场景 6）
- `DESTROY` → `CONFIRM DESTROY`：清除文明（→ **第一幕结局 B**）

### 场景 6：沟通（`scene: communication`）

**叙事**
数小时。极光几乎没有离开终端，眼睛布满红血丝。暴熊把军用水壶递给她。火花每隔一会儿问"好了吗？"，被暴熊的眼神制止了三次。磐石在门口站岗，寸步未离。

终端出现第一批可辨识的语义：
*"高维……古神……光之源……从何处来……内有何物……"*

极光靠在椅背上，闭上眼睛："它们把我们当作神明。因为对它们来说，我们的飞船就是它们的整个宇宙。"

翻译继续输出：*"我们的探测器触碰了神的皮肤……我们渴望穿越宇宙边缘……"*

外壁切口不是攻击，是它们的探测器。它们的扩张方向，是穿越飞船外壁，前往"神的宇宙"。

### 【第一幕关键选择】

#### 选择 A：`TREATY` — 签署《C区协议》

**叙事**
极光将协议草案翻译为电磁脉冲：① 废弃硬盘阵列作为新殖民地；② 提供低功率稳定能源；③ 停止穿透外壁行为。

十七分钟沉默后，微型城市的所有灯光同时熄灭一秒，又全部点亮——

*"我们接受。我们的文明将成为圣殿的守护者，如同血脉守护身躯。"*

极光把脸埋进双手里，说了一句法语。火花用拳头砸了磐石的肩膀，磐石没有躲开。暴熊在航海日志里认真地写下了几行字。

**[效果]**

## 添加新情节的格式参考

在代码中添加新场景时，遵循以下模式：

```typescript
// 在对应的 act 文件里添加新场景处理块
if (scene === '新场景名') {
  if (matchAny(full, ['触发词1', '触发词2'])) {
    return {
      messages: [
        { type: 'narrative', text: '叙事文字' },
        { type: 'system', text: '系统信息' },
        { type: 'warning', text: '警告/重要信息' },
        { type: 'lore', text: '世界观/日志内容' },
        { type: 'bond', text: '人际关系变化' },
        { type: 'success', text: '成功/进展' },
      ],
      stateChanges: { scene: '下一场景', someFlag: true },
      crewChanges: { aurora: { bond: 60, stress: 10 } },
    };
  }
}
```

**消息类型速查**
| 类型 | 用途 | 显示风格 |
|------|------|---------|
| `narrative` | 叙事散文 | 普通白色 |
